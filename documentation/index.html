Installation

Installation procedure:

  1. At first you should install Joomla on your server.
  2. Log in to your Admin Panel.
  3. Click on Extensions > Extension manager.
  4. Then select e4jdrivers_template.zip file and click Upload & Install.
  5. Once that you have installed the template you should start to setup it following the online documentation.
